# Python Basics Guide for Junior Developers

## 1. Introduction to Python
Python is a high-level, interpreted programming language known for its readability and simplicity. It's widely used in various domains such as web development, data analysis, artificial intelligence, and more.

## 2. Installing Python
To get started with Python, download and install the latest version from the official Python website: [python.org](https://www.python.org/downloads/).

## 3. Writing Your First Python Program
You can write your first Python program directly in an IDE like PyCharm, or in a simple text editor. Create a new file called `hello.py` and add the following code:
```python
print("Hello, World!")
```
Run it using the command:
```bash
tpython hello.py
```

## 4. Python Syntax
- **Variables**: Used to store information.
    ```python
    x = 5
    y = "Hello"
    ```
- **Data Types**: Common data types in Python include:
    - Integer (int)
    - Floating point (float)
    - String (str)
    - List (list)
    - Dictionary (dict)

## 5. Control Flow
- **If statements**:
    ```python
    if x > 0:
        print("Positive")
    else:
        print("Non-Positive")
    ```
- **For loops**:
    ```python
    for i in range(5):
        print(i)
    ```

## 6. Functions
Define a function using the `def` keyword:
```python
def greet(name):
    return f"Hello, {name}"
```

## 7. Conclusion
This guide covered the basics of Python to help you get started. There is much more to learn, so explore Python's documentation and resources for further study.
